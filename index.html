Welcome to Teo's digital patch.

Current projects

TEma - A stack-based (virtual (for now)) computer.
TEmu - An emulator to run TEma.
TEas - An assembler (assembly compiler) that converts TEma assembler code (TEa files) into TEma machine code objects (TEo files).
TEos - Useful programs written in TEa, eg. TExt, a text editor (WIP) and TEgn a (WIP) drawing program.
