TeMa stack based computer

TeMa is a Turing complete stack based computer definition (a virtual machine or VM) inspired by an interest in machine (code) level design, the stack based language Forth and the UXN project by Devine Lu Linvega.

TeMa grew naturally from my earlier development of a Game Boy emulator and a desire to make something from scratch.
